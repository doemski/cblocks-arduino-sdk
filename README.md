# cblocks-arduino-sdk

Arduino SDK for [cBlocks](https://github.com/informatik-mannheim/cblocks).

## Installation

Example platformio.ini

```
[env:feather]
platform = espressif32
framework = arduino
board = featheresp32

lib_deps =
  https://github.com/informatik-mannheim/cblocks-arduino-sdk
```
